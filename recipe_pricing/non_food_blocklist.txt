# R12 — last-line non-food safety gate.
#
# Normal fix path:
#   1. Move a bad SKU to the correct canonical_path in
#      reclassify_canonical_paths.py.
#   2. Re-encode HTC so htc_code/htc_form_code/htc_full_code agree with that
#      corrected path.
#
# This file is only for broad, proven non-food leakage that still reaches a
# food path after source reclassification. Do not add a phrase here when the
# SKU can be moved to a specific Non-Food path instead.
#
# Rationale: the LLM SKU classifier keeps placing cleaning/garden/personal-
# care products at food paths (Mrs Meyer's at "Pantry > Spices > Basil",
# Bonnie Plants at "Snack > Candy > Mints", Magic Man Chili Plant-Based at
# "Pantry > Oil"). The source DB should be fixed first; this catches residual
# leakage before concept_index can price recipes with it.
#
# Format: one phrase per line, blank lines & "#"-comments ignored.
# Phrases are matched as substrings (after normalization: lowercase,
# punctuation→space, multi-space collapsed). Stem-aware: "plants" matches
# "plant" too.
#
# DO NOT add "spray" alone — it has many cooking uses (cooking spray,
# vegetable spray). Add specific anti-food spray phrases instead.
